Пример программы для этого сайта:

<!DOCTYPE html>//Тип документа для браузера
<html lang="ru">//Самый главный тег
<head>//содержит метаинформацию о HTML-странице.
<meta charset="UTF-8">//Кодировка страницы
<meta name="viewport" content="width=device-width, initial-scale=1.0">//Для красивого отображения на мобильных устройствах
<title>Обучение CSS</title>//название страницы.
<link rel="stylesheet" href="style2.css">
</head>
<body>//определяет тело документа и является контейнером для всего видимого содержимого, такого как заголовки, абзацы, изображения, гиперссылки, таблицы, списки и т. д.
